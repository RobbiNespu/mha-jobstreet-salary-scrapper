MHA Jobstreet Programming language salary report 2017

What this page is showing?

this page is showing all the salaries that comes up if you search for a specific term. ofcourse this way of visualizing of the data is not accurate. for example for php you can find salaries that are over 24k MYR but when you dig in and try to find what kind of job needs php, you end up with jobs like CTO or data scientist that one part of requirment is php. but it's not really related to php or you by knowing php you can't end up on qualifying for that job. for example knowing php doesn't lead you to have a data scientist job.
anyway if you wonder what job lead to this much of salary, you can visit the github repository and view the content of report folder. all the jobs and related salaries are stored in the json format.

How to use the charts

lets first learn about the axis:
the x axis: showing the salary range, since jobstreet is showing the salaries in range, i decided to keep it as it is.
the y axis at left side: it is related to salary and it's in MYR

The histogram chart: it's related to histogram only, the histogram is showing how many time a salary range observed in the report. it comes handy to know what is the most of the salaries for specific technology.
